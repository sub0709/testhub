1. 프로그램을 개발하면 많은 라이브러리를 이용하게 되는데 
Front-End에서 흔히 사용하는 대표적인 라이브러리로 Axios가 있다.

Axios를 설치해볼것
https://www.npmjs.com/package/axios 에 설치방법과 사용방법이 설명되어있고 구글이나 네이버에서
'node axios'로 검색해도 쉽게 찾을수 있음

2. Axios로 https://www.2xl.kr/api/cardsms/group/category 을 호출하면 JSON 형태의 데이터를 받을수 있음
데이터는 아래와 같은 형태로 구성되어있다
{
	isError : 에러여부,
	msg : 에러가 있을 경우 메세지,
	data : : 데이터
}

data에는 group1, group2 를 데이터로 하는 객체의 배열로 구성되어있는데 아래와 같이 출력되도록 만들것
{ 
	'간편/인스턴트': [ '면', '가루', '진공포장', '통조림', '가공', '라면', '과자', '냉동' ],
  '과일/채소': [ '채소', '과일' ],
  '기타': [ '쌀', '할인/포인트', '상품권', '기타' ],
  '리빙': [ '화장실', '주방', '세탁' ],
  '술/음료': [ '물', '탄산/쥬스', '커피', '맥주', '소주/막걸리', '양주', '중국술', '기타' ],
  '앙념소스': [ '간장/고추장', '소금/설탕', '마요네즈/케첩', '겨자/와사비', '기타' ],
  '정육/생선': [ '돼지고기', '소고기', '닭고기', '계란', '해물', '생선' ] 
}

3. 날짜 문자열 '20200430' 를 YYYY.MM.DD 형태로 출력하는 함수를 만들것. 시간/날짜를 제어하는 라이브러리 사용 금지
예) 20200430 => 2020.04.30 으로 출력

4. 3번 함수에 옵션을 추가하여 / 또는 - 으로도 출력 가능하도록 만들것
예) 20200430 => 2020/04/30
    20200430 => 2020-04-30

5. 아래와 같은 데이터가 있을때
{
	nv_mid : 10552490824,
	cat_id : 50001313,
	frm : 'NVSHATC',
	query : '전자렌지 선반 4단'
}
위 데이터를 serialze하는 함수를 만들것. serialize하면 아래와 같은 형태가 됨
nv_mid=10552490824&cat_id=50001313&frm=NVSHATC&query=전자렌지%20선반%204단
